Git

Git on versioonihaldustööriist, mis võimaldab meil hallata meie poolt loodud koodi ja selle koodi ajalugu. Seda kasutavad kõik programmeerijad tänapäeval Git on süsteem, mis võimaldab hallata enda loodud lähtekoodi ning selle ajalugu. See on nö kohalik tööriist, programm, mille saab laadida oma arvutisse, kus saab koostada koodi (koodide hoidla), mille saab üles laadida Githubi. See tähendab, et kõik Gitiga loodud projektid saab üles panna (lükata) Githubi. Kuid oma projekti Githubi panemine ei ole kohustuslik, oma projektiga võib töötada vaid ka oma arvutis. Githubile on ka alternatiive (nt AWS Code).

Github

Github on veebipõhine koht koodiprojektide hoidmiseks (ehk kaughoidla), mille eesmärk on Giti poolt loodud projektide majutamine. Github võimaldab ka teistel inimestel (nt kolleegidel vm) teie loodud projektidele ligi pääseda

Giti käsud

git init

selle käsuga me algatame uue Git-projekti (repository) ehk lisame faili

git status

see käsk näitab koodi olekut

git add .

see käsk lisab kõik projekti kausta failid Githubi kaughoidlasse ja jälgib kõiki failiga seotud muudatusi

git commit -m '...'

see käsk teeb otsetee muudatustele, soovitatav lisada kommentaar, mis muudatus sai tehtud.

git log

selle käsuga saame kontrollida ja saada ülevaate oma kohustustest

git checkout …

selle käsuga saab kontrollida erinevaid Giti kohustusi ning see käsk aitab meil viimase juurde tagasi minna

git checkout -b …

käsk uue funktsiooni haru lisamiseks

git merge

käsk uue funktsiooni haru ühendamiseks varasemaga

git branch

käsk giti uue funktsiooni haru loomiseks

git branch -D

see on käsk giti funktsiooni haru kustutamiseks

git pull

seda käsku kasutatakse Githubi kaughoidlast sisu toomiseks ja allalaadimiseks ning kohaliku hoidla koheseks värskendamiseks selle sisuga
git push

selle käsuga saab enda loodud koodiprojekti saata Githubi kontole (kaughoidlasse)

git clone

selle käsuga saame kopeerida VisualStudio Code terminali Githubi kontolt endale vajalikku sisu

git fetch

selle käsuga saab Githubi kaughoidlast endale sisu tõmmata
